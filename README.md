# Учебный сервис прогноза погоды

Простейший сервис, который загружает исторические погодные данные, формирует признаки, обучает модели и выдаёт прогнозы.

## Что есть
- Автоматическая загрузка данных из Open-Meteo
- Предобработка данных: лаги, скользящие средние, временные признаки
- Три вида моделей: табличная, статистическая и последовательная
- FastAPI backend с базовыми эндпоинтами для прогноза, метрик и данных
- Простейший фронтенд с графиком
- Dockerfile и docker-compose

## Запуск локально

1. Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

2. Запустите сервер:

```bash
uvicorn app.api.app:app --reload
```

3. Откройте в браузере:

```
http://127.0.0.1:8000/
```

## Запуск через Docker

```bash
docker compose up --build
```

## Эндпоинты

- `GET /cities` — список доступных городов
- `GET /models` — список моделей
- `POST /forecast` — прогноз
- `GET /data` — данные по городу
- `POST /refresh` — обновление исторических данных

## Примечания

Для `SequenceForecastModel` требуется TensorFlow. Если вы не хотите устанавливать `tensorflow`, можно заменить модель на `MLPRegressor` или убрать поддержку последовательной модели.
